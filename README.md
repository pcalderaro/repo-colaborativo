# repo-colaborativo

Descripción del repositorio: Implementaremos un Behaviour Tracker en Python para registrar, analizar y hacer un seguimiento de hábitos.
Nuestro objetivo es ayudar a los usuarios a encontrar patrones en sus comportamientos, especialmente aquellos relacionados con hábitos dañinos o adicciones.

Nombre de todos los integrantes del grupo: Pilar Calderaro, Camila Iglesias, Uma Rodriguez Videla y Valentina Entrala.

Cualquier información adicional que resulte explicativa para el repositorio: Los objetivos principales serán registrar comportamientos o hábitos, analizar la frecuencia e intensidad de dichos comportamientos, identificar patrones o tendencias y finalmente proporcionar métricas que ayuden en la toma de decisiones para mejorar hábitos.
